# Retail Sales Analysis Dashboard

## Project Overview

This is an end-to-end Data Analyst project developed using Python, SQL, and Power BI. The project analyzes retail sales data, performs data cleaning, SQL analysis, and creates an interactive dashboard for business insights.

## Tools Used

* Python (Pandas, NumPy)
* MySQL
* Power BI
* DAX

## Project Workflow

1. Data Cleaning using Python
2. Data Analysis using SQL
3. Dashboard Development using Power BI
4. KPI Creation using DAX Measures

## Dashboard Preview

![Dashboard](dashboard.png)

## Dashboard Features

* Total Sales
* Total Profit
* Total Orders
* Average Sales
* Profit Margin %
* Sales by Category
* Sales by State
* Sales by Month
* Sales by Region
* Category Slicer
